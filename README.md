# ЖМ «Жан жылуы» — статический сайт (демо)

Короткий статический шаблон сайта балабақша на казахском языке. Использует данные и название с исходного сайта для наполнения контента.

Как запустить локально:

1. Откройте файл `index.html` в браузере (двойной клик или `open`).
2. Или запустите простой HTTP-сервер из папки `zhanzhyluy-site`:

```bash
cd zhanzhyluy-site
python3 -m http.server 8000
# затем откройте http://localhost:8000
```

Что сделано:
- Современная адаптивная верстка (HTML + CSS)
- Секция "Біз туралы", "Қызметтер", "Жаңалықтар", "Фотосуреттер", "Байланыс"
- Использованы текстовые данные с оригинального сайта (название, директор, email, краткие описания)
